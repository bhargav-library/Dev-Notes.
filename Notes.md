Variable: a variable is a named storage location in a computer's memory that holds a value. It acts as a place holder for data, allowing us to store, retrive and modify information within a program.

A data type specifies the kind of values a variable can hold and the operations that can be performed on those values.
common data type include: 
i) integers : for whole numbers without decimal points.

ii) floating : for numbers with decimals.

iii) characters : for single letters or symbols.

iv) booleans : for true/ false values.

v) string : A string represents a sequence of characters. It is used to store and manipulate texted data. Internally a string is usually implemented as an array of characters, but it is treated as a single data type with bult in operation like length, concatenation and substring.
